### Versionsverwaltung ###
Ein typisches Einsatzgebiet von Versionsverwaltungssystemen befindet sich im Umfeld der Software-Entwicklung, um Quellcode an den mehrere Personen arbeiten zu organisieren. Es lassen sich die Änderungen an jeder einzelnen Datei über die Zeit nachverfolgen.
Einige dieser Versionsverwaltungssysteme sind auch als Open Source Software verfügbar.
Ein wichtiges Merkmal der Versionsverwaltung liegt in der Koordinierung des gemeinsamen Zugriffs auf eine Datei.
